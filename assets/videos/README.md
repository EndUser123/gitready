# NotebookLM Videos

This directory should contain videos downloaded from NotebookLM.

## Instructions

1. Open your NotebookLM notebook for `gitready`
2. Generate Video Overviews in Studio
3. Set notebook sharing to "Anyone with the link"
4. Click Share on the video and copy the link
5. Download the MP4 version as backup
6. Place downloaded videos in this directory:
   - `github-ready_explainer_video.mp4` (22-second overview)
   - `github-ready_explainer_podcast.mp4` (2m 20s deep dive)

## Usage in README.md

```markdown
[![Watch the video](assets/infographics/gitready_architecture.png)](https://notebooklm.google.com/video/YOUR_VIDEO_ID "Full video overview")

**📺 Video Overview** | **⬇️ Download MP4**
```

## Why This Approach?

- ✅ NotebookLM hosts reliably (Google infrastructure)
- ✅ Thumbnail → video link pattern works everywhere on GitHub
- ✅ Optional MP4 backup for offline access
- ✅ Follows 2026 best practices for GitHub README videos
